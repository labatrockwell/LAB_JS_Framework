dispatchEvent(typeOrEvent, optionalParams)

There are 2 ways to use dispatchEvent. The first is to send an Event object (including objects that extend the Event class) and any optionl parameters. For example:
Parameters:
Name Type Description
typeOrEvent string | Event The event type as a string or an Event object
optionalParams Object Optional parameters passed as an object. return {boolean} True if dispatch is successful, false if not.
Inherited From:
  • LAB.EventDispatcher#dispatchEvent

Examples

var params = {secret: "my secret", message: "hi"};
